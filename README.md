# rabbit_mq_prototype

#### Run RabbitMq locally on docker
```docker
docker run -it --rm --name rabbitmq -p 5672:5672 -p 15672:15672 rabbitmq:3.11-management
 ```
 
 [RabbitMq installation guide](https://www.rabbitmq.com/download.html)
